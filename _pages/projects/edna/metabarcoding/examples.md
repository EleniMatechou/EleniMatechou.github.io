## R script

This script analyses simulated metabarcoding data with:

- n = 100 sites  
- S = 10 species  
- M = 5 samples  
- K = 12 qPCR tests  

👉 [R script](https://www.dropbox.com/scl/fi/vti57ww96iqg5m92khm6k/eDNAPlus.html?rlkey=1e24mq96xbproewj0zyozywrf&dl=0)

## How to use

1. Download all files  
2. Save them in your working directory  
3. Run the R script to:
   - install required packages  
   - fit the model  
   - generate outputs and plots
